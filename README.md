### Requirements
- Ruby: 3.0.1
- Rails: 6.1.4
- Postgresql: 13.3

### How to run

1. `bundle install`
2. `rails db:create`
3. `rails db:migrate`
4. `rails s`

### How to use

#### Admin
- Access to: http://localhost:3000/admin for admin page

#### APi
- Access to: http://localhost:3000/api for API

### How to test
- Run `bundle exec rspec` to run all unit tests
